# lecturelinker
Setting up
---------------------
1. Install pip or pip3
2. Install flask
```
$ pip install Flask
```
Or
```
$ pip3 install Flask
```
3. Install playwright
```
$ pip install playwright
```
Or
```
$ pip3 install playwright
```

Running the webserver
---------------------
Run app.py with python(3):

```
$ python app.py
```
Or
```
$ python3 app.py
```
